Structure globale de la chaîne d'effet
======================================

[lfo 1] -> | [mic]
[lfo 2] -> | | \
[randi] -> | | [loop 1] - [loop 2] - [loop 3] - [loop 4]
[randh] -> | | /
           | [2 stereo delays (parallel or serial)]
[control   | |
 routing   | [disto + simple lowpass]
 matrix]   | |
           | [flanger]
           | |
           | [moog resonant lowpass filter]
           | |
           | [amplitude modulation (off, am, ring)]
           | |
           | [4 bands EQ]
           | |
           | [compressor]
           | |
           | [audio output]
